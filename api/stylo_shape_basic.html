class stylo.shape.basic.Ellipse(x, y, a, b, r)[source]¶

Bases: stylo.shape.shape.Shape

An ellipse.

Mathematically we can define an ellipse to be the set of points \((x, y)\) which satisfy:

\[\frac{(x - x_0)^2}{a^2} + \frac{(y - y_0)^2}{b^2} \leq r^2\]

where:

  • \((x_0, y_0)\) is the center of the ellipse
  • \(a\) is known as the semi major axis, larger values make the ellipse more elongated in the \(x\)-direction
  • \(b\) is known as the semi minor axis, larger values make the ellipse more elongated in the \(y\)-direction
  • \(r\) is the “radius” of the ellipse and controls the overall size of the ellipse
