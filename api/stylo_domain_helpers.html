Stylo.Domain.Helpers¶

class stylo.domain.helpers.CartesianConversion[source]¶

Bases: object

A helper class for defining new domain classes.

Say you are implementing a new class, perhaps the Rotation domain transform where the interesting part is the adjustments to the r and t coordinates. The implementation of x and y is then done with respect to the transformed r and t variables.

This class provides implementations of the _get_xs and _get_ys methods to handle this conversion for you automatically, leaving you to focus on implementing the transform that interest you.

To use this class, simply include it in your class definition as follows

class MyDomain(CartesianConversion, RealDomain):

    def _get_xs(self):
        ...

    def _get_ys(self):
        ...

Note

The order in which the classes are listed is very important.

_get_x()[source]¶
