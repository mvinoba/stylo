Stylo.Testing._factory¶

This module contains functions that automate the process of writing base test classes for Domain objects.

stylo.testing._factory.define_domain_test(domain)[source]¶

Given a domain definition, write the test class that will automatically verify that a given subclass follows the expected interface.
