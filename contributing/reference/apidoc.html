Apidoc.py¶

This script aims to provide similar functionality to the sphinx-apidoc command but produce results with better integration with the way the documentation is structured.

Usage¶

As it currently stands this script depends only on the Python standard library. If you are making changes to the script and want to run it in isolation then it can be done as follows:

$ python scripts/apidoc.py -m <path_to_module> -o <path_to_output>
