Number Systems¶

There are a number of different collections (called sets) of number that are in use all over mathematics.

Natural Numbers¶

The so called counting numbers, the set of natural numbers contains all the positive whole numbers and is referred to by the symbol \(\mathbb{N}\).

\[\mathbb{N} = \{0, 1, 2, 3, 4, 5, \ldots\}\]

Note

There is some debate over whether \(0\) should be considered a natural number. So you may see the symbol \(\mathbb{N}_0\) when someone wants to be explicit that they consider \(0\) to be a natural number.

Integers¶

The integers is the set of all positive and negative whole numbers including 0 and is referred to by the symbol \(\mathbb{Z}\)

\[\mathbb{Z} = \{ \ldots, -2, -1, 0, 1, 2, \ldots \}\]
